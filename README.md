# 🐄 Livestock Tracker - Government Edition

A production-ready livestock tracking system with GPS geofencing, health records, and government compliance reports.

## 📁 Project Structure
```
animal-tracker-rn/
├── backend/        # Flask REST API
└── mobile/         # React Native (Expo) App
```

## 🚀 Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Mobile
```bash
cd mobile
npm install
npx expo start
```

## 🌐 Deployment
- Backend: Render.com / Railway.app
- Mobile: Expo Go / EAS Build

## 📱 Features
- GPS Animal Tracking
- Geofence Alerts
- Health Records
- Government Reports
- Multi-farmer Support
